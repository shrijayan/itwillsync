How it works

    


      

        

          
        

        
One command, zero install

        
npx itwillsync -- claude and you're running. No global install, no config files.

      


      

        

          
        

        
QR code connect

        
A QR code appears in your terminal. Scan it on your phone. Connected.

      


      

        

          
        

        
Full bidirectional terminal

        
Not just a viewer. Type on your phone, see it on your laptop. Control your agent from anywhere.

      


      

        

          
        

        
Local WiFi or Tailscale

        
Same WiFi by default. Add Tailscale to connect from anywhere — auto-detected, one-time setup.

      


      

        

          
        

        
Multi-device access

        
Connect from phone, tablet, and laptop simultaneously. All see the same terminal session.

      


      

        

          
        

        
Auto-reconnect

        
WiFi dropped? Phone locked? Reconnects automatically and catches up with recent output.

      


    

  


  
  

    
Quick start

    

      

        
        
# Run directly (no install needed)
$ npx itwillsync -- claude

# Or with any agent
$ npx itwillsync -- aider
$ npx itwillsync -- bash

      

      
Requires Node.js 20+. Works on macOS, Linux, and Windows.
